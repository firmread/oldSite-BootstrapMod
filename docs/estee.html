Estée Lauder's interactive product catalog app for custom touchscreen in concept store mockup. This project was made in combination of various technologies. It's wrapped in OpenFrameworks to make smooth transitions, powerful video handling, and customizable touchscreen events input. While also bring some of the best web libraries through Awesomium frameworks to make the content easily customizable in future iterations for the client.

The project is written in Awesomium Frameworks, OpenFrameworks, HTML, CSS, JavaScript.
